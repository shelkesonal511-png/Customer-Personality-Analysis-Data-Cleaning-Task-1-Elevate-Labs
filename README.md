# Customer Personality Analysis – Data Cleaning & Preprocessing

## 📌 Project Overview

This project focuses on cleaning and preprocessing the **Customer Personality Analysis (Marketing Campaign)** dataset using **Microsoft Excel** and **Python (Pandas)**. The objective was to transform raw customer data into a clean, consistent, and analysis-ready dataset by handling missing values, removing duplicates, correcting data types, and standardizing formats.

---

## 🎯 Objectives

- Identify and handle missing values
- Remove duplicate records
- Standardize text and date formats
- Validate and correct data types
- Prepare the dataset for further analysis and visualization

---

## 📂 Dataset

- **Dataset:** Customer Personality Analysis (Marketing Campaign)
- **Source:** Kaggle
- **Records:** 2,240
- **Features:** Customer demographics, income, purchasing behavior, campaign responses, and product spending.

---

## 🔧 Data Cleaning Process

### Excel

- Imported the raw dataset
- Checked for missing values
- Removed duplicate records
- Verified data consistency
- Applied appropriate formatting

### Python (Pandas)

- Loaded the dataset using Pandas
- Identified missing values
- Removed duplicate rows
- Renamed column headers for consistency
- Converted date columns to datetime format
- Verified and corrected data types
- Cleaned inconsistent values
- Exported the cleaned dataset
